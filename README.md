# 💌 실물 재현형 인터랙티브 모바일 청첩장

이 프로젝트는 실물 종이 청첩장의 3단 접지 레이아웃과 편지지 보관 포켓, 그리고 숨겨진 Q&A 쪽지 기능을 웹상에 인터랙티브하게 재현한 커스텀 모바일 청첩장입니다.

## 🌟 주요 인터랙티브 기능
1. **편지 봉투 개봉 애니메이션:**
   * 처음 페이지에 접속하면 제비 모양 스티커로 밀봉된 빈티지 편지 봉투가 나타납니다.
   * 제비 스티커를 터치하면 3D Perspective 효과와 함께 봉투 좌우의 문이 회전하여 열리며 내부 카드가 공개됩니다.
2. **Q&A 슬라이딩 카드:**
   * 청첩장 내부 가운데에 날아가는 제비 그림과 함께 `To`가 쓰여진 쪽지가 꽂혀있습니다.
   * 이 카드를 터치하거나 위로 쓸어올리면(`PULL UP`), 쪽지가 위로 부드럽게 슬라이드 되면서 카드 밑에 숨겨져 있던 신랑 신부의 Q&A 문답이 나타납니다.
   * 다시 카드를 터치하면 제자리로 들어갑니다(`PUSH DOWN`).
3. **세련된 모바일 최적화 레이아웃:**
   * 스마트폰 세로 비율에 맞추어 열린 청첩장이 세로 방향(상단: 초대 문구 / 중앙: 슬라이딩 카드 / 하단: 예식 개요)으로 배치되어 뛰어난 가독성을 자랑합니다.
4. **본문 내용 페이드인:**
   * 봉투를 개봉하면 아래쪽에 카운트다운 타이머, 사진첩, 대중교통 지도 정보, 송금 계좌 정보, 참석 의사 전달(RSVP) 등 풍부한 모바일 청첩장의 본문 내용이 자연스럽게 나타납니다.

## 📂 폴더 구조
```text
wedding-custom/
├── index.html        # 메인 마크업 (실물 청첩장 구조 및 본문)
├── css/
│   └── style.css     # 청첩장 고유 스타일시트 (3D 회전 애니메이션, 종이 질감, 타이포그래피)
├── js/
│   └── app.js        # 봉투 개봉 애니메이션, 쪽지 밀고 당기기, 디데이 카운터, 계좌 복사 등
├── images/
│   └── (실제 사진을 넣을 폴더, 기본적으로는 흰색 감성 카드 템플릿으로 대체)
└── README.md         # 사용 및 배포 안내
```

## 💻 로컬에서 미리보기
터미널에서 이 폴더(`wedding-custom`)를 기준으로 다음 명령어를 실행하여 확인하실 수 있습니다.

```bash
python3 -m http.server 8000
```
웹 브라우저를 열고 **[http://localhost:8000](http://localhost:8000)**에 접속하세요. 
*개발자 도구(F12)에서 모바일 뷰로 확인하시는 것을 강력 권장합니다.*

## 🚀 GitHub Pages에 배포하기 (내 깃허브 io 올리기)

1. **GitHub 원격 저장소 생성:**
   * GitHub에서 새로운 Repository를 생성합니다 (예: `wedding-custom`).
   
2. **코드 커밋 및 푸시:**
   로컬 터미널에서 다음 명령어를 실행하여 새로 만든 GitHub 저장소에 코드를 업로드합니다.
   ```bash
   git init
   git add .
   git commit -m "feat: 커스텀 실물 재현 청첩장 완료"
   git branch -M main
   git remote add origin https://github.com/사용자이름/wedding-custom.git
   git push -u origin main
   ```

3. **GitHub Pages 활성화:**
   * 저장소 페이지의 **Settings -> Pages** 메뉴로 이동합니다.
   * **Branch**를 `main`으로 선택하고 **Save**를 누르면, 잠시 후 배포 완료 주소(`https://사용자이름.github.io/wedding-custom/`)가 생성됩니다.
